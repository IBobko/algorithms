# Линейное пространство

**Линейное (или векторное) пространство** — это математическая структура, состоящая из множества элементов (называемых векторами), для которых определены две операции: сложение друг с другом и умножение на число (скаляр). Эти операции подчиняются 8 фундаментальным правилам, которые делают пространство «привычным» для вычислений.


Линейное пространство здесь — не математика из линала, а просто структура, где элементы стоят в одну линию и у каждого есть позиция.
